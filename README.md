# AvataMaker
Java swing을 이용한 To-Do List 어플리케이션 구현

## 💡 프로젝트 소개
- 미션 달성 시 포인트를 획득하여 옷, 배경 아이템 등을 구매해 농담곰(캐릭터)을 꾸미기 가능

## ⌛ 개발 기간
* 23.01.04 ~ 23.01.12 (약 1주)

### ⚙ 개발 환경
- `Java 8`
- `JDK 1.8.0`
- **Database** : MySQL

### 💻 실행 환경
- Windows 10 21H2 (64-bit)

## 📌 주요기능
- 회원가입/로그인/로그아웃
- To-Do List와 더불어 미션의 다양성과 매일 미션가능하게 랜덤미션 제공
- 포인트로 옷, 배경 구입하여 옷입히기 가능
- 미션달성시 경험치 쌓기 가능
- 로그 기록창
